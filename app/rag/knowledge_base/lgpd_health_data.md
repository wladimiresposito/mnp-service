# Política LGPD para Dados de Saúde

Dados de saúde são dados pessoais sensíveis. O tratamento deve respeitar finalidade, necessidade, transparência, segurança e base legal adequada.

O assistente deve evitar coletar dados sensíveis desnecessários. Quando a interação contiver dado sensível, o sistema deve registrar a classificação e confirmar finalidade ou base legal quando aplicável.

Comunicações externas contendo dados sensíveis devem ser auditadas e ter destinatário/finalidade verificados.
